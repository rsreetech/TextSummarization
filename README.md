# TextSummarization

Text Summarization using spaCy or gensim in Python

Prerequisites

spaCy (https://spacy.io/)

pytextrank (https://spacy.io/universe/project/spacy-pytextrank/)

Python 3.5 and above

gensim (https://pypi.org/project/gensim/)
